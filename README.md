# Meu Portfólio

Projeto desenvolvido na unidade curricular de HTML e CSS.

Figma: https://www.figma.com/design/NBFauvA3NQZKUtneIsnlum/Portf%C3%B3lio---Henrique-Santos-TSI1?node-id=0-1&t=rIbYjTB7zU304KY8-1

## Tecnologias
- HTML5
- CSS3
- Bootstrap 5.3.0

## Como visualizar

Baixe o projeto e abra o arquivo "index.html" no navegador.

## Estrutura do Projeto

- **index.html**: Página principal do portfólio
- **assets/styles/styles.css**: Arquivo de estilos CSS personalizado

## Seções

- **Início**: Apresentação principal com avatar e botões de navegação
- **Sobre mim**: Breve apresentação pessoal e experiência
- **Habilidades**: Minhas principais competências divididas em Front-End, Design e Back-End
- **Projetos**: Exemplos de trabalhos realizados com descrições e tecnologias utilizadas
- **Contato**: Links para e-mail, GitHub e LinkedIn

## Recursos Utilizados

### Ícones SVG
Os ícones utilizados no projeto foram obtidos do **Feather Icons** (https://feathericons.com/), uma biblioteca gratuita de ícones em formato SVG. 

Ícones utilizados:
- Code (Front-End)
- Tool/Pen (Design)
- Database (Back-End)
- Mail (Email)
- GitHub (GitHub)
- LinkedIn (LinkedIn)

### Paleta de Cores
- **#FFFFFF** - Branco (fundos principais)
- **#060604** - Preto (navbar e rodapé)
- **#4F757E** - Azul petróleo escuro (seções destaque)
- **#819EAC** - Azul acinzentado claro (elementos secundários)
- **#9C8C82** - Bege/marrom acinzentado (backgrounds alternativos)

## Responsividade

O site utiliza media queries para adaptar o layout em diferentes tamanhos de tela:
- Desktop: Layout completo com elementos lado a lado
- Tablet (max-width: 768px): Elementos em coluna
- Mobile (max-width: 480px): Layout otimizado para celular

## Observações

- Utilizei Bootstrap apenas para o sistema de grid e componentes básicos
- Todos os estilos personalizados estão no arquivo `assets/styles/styles.css`
- Os ícones SVG foram copiados diretamente do site Feather Icons

- Cores e layout criados do zero seguindo a paleta de cores definida
